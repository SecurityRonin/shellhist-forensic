# Privacy Policy

`shellhist-forensic` is an offline Rust library. It performs no network
communication, collects no telemetry, and transmits no data of any kind. All
processing of shell command-history files and host artifacts happens entirely on
your own machine, under your control.

Because the software gathers nothing, there is nothing for us to store, share, or
sell. Any data you process with it remains yours alone.
